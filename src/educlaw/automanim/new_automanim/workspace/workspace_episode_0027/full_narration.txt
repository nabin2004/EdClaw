Welcome to the Dead ReLU Neuron visualization. Today, we'll explore one of the fundamental problems in deep learning. The ReLU activation function outputs zero for all negative inputs, and the input value for all positive inputs. This red shaded region is the dead zone. When a neuron's input falls here, the output is always zero. Look at this example. When x equals negative two, the output is zero. The gradient, or derivative, of ReLU is zero for all negative inputs, and one for all positive inputs. Zero gradient means no signal flows backward through the network. During backpropagation, weights are updated using the gradient. The chain rule multiplies gradients through each layer. But when the ReLU gradient is zero, the entire product becomes zero. This means the weight update term vanishes. The result? The new weight equals the old weight. Nothing changes. The neuron is now dead. Its weights are frozen forever, no matter how long you train. Remember: ReLU dies when x is less than zero, which means gradient equals zero, which means weights never update.
